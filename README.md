로또 프로그램



## 💡 프로젝트 소개
- 흡연자들이 뭉쳐서 만든, 오직 재미를 위한 개발자 키우기 게임
- 여러 가지 게임 내 활동을 통하여 LevelUp 및 재화를 획득하고 취업 시 게임 Clear

## ⌛ 개발 기간

- 22.12.07. ~ 22.12.19.

### ⚙ 개발 환경

- `Java 8`
- `JDK 1.8.0`
